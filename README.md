# TypeScript Next.js Clean Architecture

A mouthful I agree! Pretty cool though.

**Install:**

```
npm install
```

**Run development server:**

```
npm run dev
```

**Run tests:**

With development server running in another tab:

```
npm test
```

Or watch your tests:

```
npm test -- --watch
```

**Run in production mode:**

```
npm run prepare
npm start
```
